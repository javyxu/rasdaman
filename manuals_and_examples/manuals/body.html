This is the documentation of rasdaman community, the free, open-source multi-dimensional Array Analytics Server with outstanding scalability.

  • If you want to set up rasdaman then you should consult the Installation and Administration Guide; best read in conjunction with the PostgreSQL External Products Guide.
  • Retrieval and manipulation of raster data is documented in the Query Language Guide (PDF).

  • The core rasdaman system implements a domain-agnostic Array Database System implemented in C++. Client APIs, conforming to the ODMG standard, are available in C++ (see C++ Development Guide, PDF and API documentation HTML ) and Java (see Java Development Guide, PDF and API documentation HTML ).
  • The C++ sources of the rasdaman engine are documented for the complete engine, API, and the PostgreSQL base DBMS interface
  • Spatio-temporal geo services based on the OGC standards WMS, WCS, and WCPS are implemented in the petascope component of the rasdaman system. See the web service interfaces (HTML), import tools (HTML), petascope database schema for geo metadata (HTML), and the code documentation (HTML).
  • The Semantic CRS resolver (SECORE) is a geo Web service component for hosting definitions of geospatial and temporal Coordinate Reference Systems (CRSs) following the OGC Name Type Specification; see the Java class documentation (HTML).
